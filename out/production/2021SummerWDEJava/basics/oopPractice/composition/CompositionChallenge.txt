***COMPOSITION CHALLENGE***
Create a single room of a house, using composition
Think about the things that should be included in the room - This could be physical parts of the room, but furniture too
Add at least one method to access an object via a getter and then access that object's public method.

Your room must have at least 5 objects
Each classes constructor must have at least 2 parameters that it is accepting & using it to initialize the class fields
Each object(class) must have at least 2 methods

Create a main (driver) class to construct your room
